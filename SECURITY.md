# Security Policy

## Supported Versions

| Version   | Supported          |
| --------- | ------------------ |
| > 1.0.0   | :white_check_mark: |
| < 1.0.0   | :x:                |

## Reporting a Vulnerability

Please, open an issue explaining the vulnerability found.

If you know how to fix it, a PR is always welcome.

Thank you :)
